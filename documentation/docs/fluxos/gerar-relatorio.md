---

## Contexto

### Gatilho
Gestor ou professor precisa gerar relatório de desempenho para análise ou apresentação.

### Pré-condições
- Usuário com permissão de relatórios
- Dados de progresso existentes
- Período selecionado

### Resultado Esperado
- Relatório gerado em formato desejado (PDF, Excel)
- Dados consolidados por turma/aluno
- Métricas de engajamento e desempenho

---

## Tipos de Relatório

| Tipo | Público | Dados |

| Aluno | Professor/Responsável | Detalhamento individual |
| Escola | Gestor | Consolidado de turmas |
| Rede | Network Manager | Consolidado de escolas |
