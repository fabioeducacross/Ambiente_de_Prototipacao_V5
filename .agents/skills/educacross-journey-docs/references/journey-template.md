## 🔤 Tom e Linguagem

### ✅ Faça

- Use **2ª pessoa do singular** ("você clica", não "o usuário clica")
- Verbos no **imperativo** para instruções ("Clique", "Preencha", "Confirme")
- **Negrito** para elementos de UI ("botão **Salvar**")
- Termos técnicos com **explicação inline** primeira vez
- Linguagem **clara e direta** (evite jargões desnecessários)

### ❌ Evite

- Voz passiva ("o botão deve ser clicado")
- Termos ambíguos ("clique ali", "vá até lá")
- Suposições de conhecimento ("como você já sabe...")
- Humor excessivo (piadas podem confundir)
- Abreviações não explicadas (TLA = Three Letter Acronym)

### Exemplo Comparativo

❌ **Ruim**:
```
Após ter feito login no sistema, o usuário deverá proceder 
à criação de uma nova atividade clicando no menu lateral.
```

✅ **Bom**:
```
❶ Faça login na plataforma  
❷ No menu lateral esquerdo, clique em **"Atividades"**  
❸ Clique no botão **"+ Nova Atividade"** (canto superior direito)
```

# 🎯 Criar Atividade de Missão

> **Persona**: Professor  
> **Duração estimada**: 8-12 minutos  
> **Última atualização**: 2026-02-12

---

## 📝 Objetivo

Aprenda a criar uma nova atividade do tipo **Missão** para seus alunos, desde a definição do título e objetivos de aprendizagem até a publicação com data programada.

**Ao final desta jornada, você será capaz de:**
- ✅ Criar atividades de Missão com objetivos claros
- ✅ Configurar datas de início e término
- ✅ Anexar arquivos e recursos multimídia
- ✅ Publicar imediatamente ou agendar para data futura

---

## ✅ Pré-requisitos

Antes de começar, certifique-se de que:

- [ ] Você está logado como **Professor**
- [ ] Você tem pelo menos 1 turma ativa
- [ ] Você tem permissão de **criar atividades**
- [ ] Material didático está preparado (opcional)

---

## 📸 Passo a Passo

### Passo 1: Acessar o Calendário de Atividades

![Dashboard Professor](./screenshots/prof-001-001-dashboard.png)

**Instruções:**

❶ No menu lateral esquerdo, clique em **"Calendário"**  
❷ Observe a visualização mensal do calendário  
❸ Clique no botão **"+ Nova Atividade"** (canto superior direito)

:::tip Atalho de Teclado
Pressione `Ctrl + N` (Windows) ou `Cmd + N` (Mac) para criar atividade rapidamente.
:::

---

### Passo 2: Preencher Informações Básicas

![Formulário de Atividade](./screenshots/prof-001-002-formulario.png)

**Instruções:**

❶ No campo **"Título"**, digite um nome descritivo (ex: "Missão: Exploração Científica")  
❷ Selecione **"Missão"** no dropdown **"Tipo de Atividade"**  
❸ No campo **"Descrição"**, descreva os objetivos (mínimo 20 caracteres)  
❹ Clique em **"Próximo"** para continuar

:::caution Atenção
O título não pode ser alterado após publicação. Revise com cuidado antes de prosseguir.
:::

---

### Passo 3: Configurar Datas e Turmas

![Configuração de Datas](./screenshots/prof-001-003-datas.png)

**Instruções:**

❶ No campo **"Data de Início"**, selecione a data de liberação  
❷ No campo **"Data de Término"**, selecione o prazo final  
❸ Na lista **"Turmas"**, marque as turmas que farão a atividade  
❹ Clique em **"Próximo"**

:::info Validação de Datas
A data de término deve ser pelo menos 1 dia após a data de início. Atividades com prazo < 24h não podem ser criadas.
:::

---

### Passo 4: Anexar Materiais (Opcional)

![Anexar Arquivos](./screenshots/prof-001-004-anexos.png)

**Instruções:**

❶ Clique no botão **"Adicionar Arquivo"**  
❷ Selecione arquivos do seu computador (PDF, DOCX, JPG, PNG)  
❸ Aguarde o upload (barra de progresso verde)  
❹ Repita para anexar múltiplos arquivos  
❺ Clique em **"Próximo"**

:::tip Formatos Aceitos
Aceitos: PDF, DOCX, PPTX, JPG, PNG, MP4 (até 50MB cada)  
Não aceitos: EXE, ZIP, RAR
:::

---

### Passo 5: Revisar e Publicar

![Revisão Final](./screenshots/prof-001-005-revisao.png)

**Instruções:**

❶ Revise todas as informações na tela de resumo  
❷ Escolha uma das opções:
   - **"Publicar Agora"** → Atividade fica imediatamente disponível
   - **"Agendar"** → Atividade será publicada na data de início configurada
❸ Clique no botão escolhido
❹ Aguarde a mensagem de confirmação verde

:::success Atividade Criada!
Sua atividade foi criada com sucesso. Os alunos das turmas selecionadas verão a atividade no calendário deles.
:::

## ⚠️ Casos de Erro

### Erro: "Título já existe para esta turma"

**Causa**: Já existe uma atividade com este título exato para uma das turmas selecionadas.

**Solução**:
1. Volte ao Passo 2
2. Modifique o título adicionando um número ou data (ex: "Missão Científica - Fev/2026")
3. Continue o processo normalmente

**Prevenção**: Use títulos descritivos e únicos incluindo data ou número da aula.

---

### Erro: "Data de término inválida"

**Causa**: A data de término é anterior ou igual à data de início.

**Solução**:
1. Volte ao Passo 3
2. Certifique-se de que a data de término seja pelo menos 1 dia após a data de início
3. Se precisar de atividade de 1 dia, coloque início às 00:00 e término às 23:59

---

### Erro: "Arquivo muito grande para upload"

**Causa**: O arquivo excede o limite de 50MB por arquivo.

**Solução**:
1. Comprima o arquivo usando ferramentas como WinRAR, 7-Zip (para PDFs: use Adobe Acrobat)
2. Ou divida o conteúdo em múltiplos arquivos menores
3. Ou hospede o arquivo em Google Drive/OneDrive e cole o link na descrição

**Prevenção**: Otimize vídeos antes do upload (720p, codec H.264).

## 📊 Métricas e Validação

**Como validar que a atividade foi criada corretamente**:
- [ ] Atividade aparece no calendário do professor (mês correto)
- [ ] Atividade está visível para alunos das turmas selecionadas
- [ ] Tipo de atividade está como "Missão" com ícone de alvo (🎯)
- [ ] Datas de início e término estão corretas
- [ ] Arquivos anexados abrem normalmente

**Indicadores de sucesso**:
- Tempo médio de conclusão: **10 minutos**  
- Taxa de sucesso na primeira tentativa: **85%**  
- Erros mais comuns: 
  - Título duplicado (45%)
  - Data inválida (30%)
  - Arquivo muito grande (15%)
