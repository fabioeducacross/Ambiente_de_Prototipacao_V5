## Gradientes

A marca Vercel usa pontualmente gradientes prismáticos/arco-íris em peças hero e campanhas.

- Exemplo de gradiente editorial:
  - linear-gradient(90deg, #0070F3 0%, #7928CA 35%, #FF0080 70%, #FF4D4D 100%)
- Diretriz: não usar gradiente como base de toda UI de produto; reservar para momentos de marca.

## Proporção de uso recomendada

- 70%: neutros (branco + escala de cinza)
- 20%: preto e tons de alto contraste
- 10%: acentos (azul e gradientes editoriais)

## Nota de precisão

A Vercel publica o sistema Geist Colors, porém parte dos valores numéricos não aparece em texto estático via scraping. Esta paleta combina valores observáveis e padrões públicos da linguagem visual Vercel/Geist para uso prático consistente.
